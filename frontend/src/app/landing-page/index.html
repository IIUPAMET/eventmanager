About PEM

Event manager is a great system of managing your events.
We have prepared for you a really large list of pre-installed events. You can sign in by Google account, find friends, communicate and of course create new beautiful events. The event creation module allows you to accurately determine the date and time of the event and the end of the event. Integration Google Maps service gives you the ability to specify where they occur.
